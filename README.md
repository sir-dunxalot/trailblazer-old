## Installation

- `npm install -g firebase-tools`
- `git clone git@github.com:sir-dunxalot/trailblazer.git` this repository
- `cd trailblazer`
- `npm install`
- `bower install`
- `firebase login`

## Running / Development

### Locally:

* `ember server`
* Visit your app at [http://localhost:4200](http://localhost:4200).

### On Production:

- `ember firebase deploy`
- `firebase open`
